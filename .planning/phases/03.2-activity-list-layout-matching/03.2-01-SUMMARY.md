# Summary: 03.2-01 Portal Theme CSS Infrastructure & Activity List Grid Styling

## One-Liner
Reusable portal-themed CSS system with legacy-matching activity list grid styling, filter toggle, and visual polish.

## What Was Built
- **Portal theme CSS infrastructure**: Three CSS files — `_grid-base.css` (shared grid via CSS custom properties), `theme-recruitingportal.css` (teal), `theme-adportal.css` (purple stub)
- **Legacy-matching grid styling**: Tahoma 14px font, 11px 13px cell padding, #e6f6f6 header backgrounds, white cells, 7px solid border frame, no box shadow
- **Filter toggle**: Column filters hidden by default, toolbar funnel icon toggles visibility with Danish labels
- **Visual polish**: White badge for count, lighter hover colors (#f2fbfa/#f7f5f9), bold column headers, always-visible sort icons, inline-flex header alignment, dark green progress bar
- **ColGroup column widths**: Headline flexible fill, fixed widths for other columns
- **Theme wrapper**: MainLayout wraps content in `theme-recruitingportal` class
