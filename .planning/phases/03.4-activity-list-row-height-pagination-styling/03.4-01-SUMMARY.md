## Accomplishments
- Activity list body rows reduced to 19px height (from ~36px) via --portal-grid-row-height, --portal-grid-cell-line-height, and reduced cell padding
- Pagination footer background matches grid header (#e6f6f6 for recruiting, #f0edf2 for ad portal)
- Header row height unchanged -- still controlled by --portal-grid-header-padding: 11px 13px
- Both recruiting and ad portal themes updated with new CSS custom properties

## Files Created/Modified
- `src/SignaturPortal.Web/wwwroot/css/portal-themes/theme-recruitingportal.css` - Added --portal-grid-row-height: 19px, --portal-grid-cell-line-height: 17px, changed cell padding to 1px 13px
- `src/SignaturPortal.Web/wwwroot/css/portal-themes/theme-adportal.css` - Same three property changes as recruiting theme
- `src/SignaturPortal.Web/wwwroot/css/portal-themes/_grid-base.css` - Added body row height rule, cell line-height/height/overflow, pagination background and toolbar styling

## Decisions Made
- 1px vertical cell padding (down from 11px) combined with 17px line-height achieves exact 19px: 1px top + 17px line + 1px bottom = 19px
- Height set on both `<tr>` and `<td>` elements for cross-browser compatibility
- overflow:hidden on cells prevents text from expanding rows beyond 19px
- Pagination toolbar gets its own rule with themed border-top (replaces MudBlazor default gray)
- Fallback values (auto, normal) in var() ensure backward compatibility for themes without new properties
